Our second assignment for “Interactive Media: Web” was to create a web page which could embody a random word given to us. The word I was given was “Time”. Another thing to note, this was our first assignment where we were to explore the possibilities of javascript.

To accomplish my goal of embodying time, I chose to create pendulum physics by applying triangle math and rendering the results through an SVG generated by the script. The math for the physics are based off of Daniel Shiffman’s example done in processing.

There are four of these pendulums on the web page, each in their own individual cells. When a pendulum reaches the centre of its cell, it plays back a synthesizer tone based on a note of the user’s choice. This turns each pendulum into a metronome which, when combined together, create complex and evolving poly-rhythm. Jean-Michel Basquit can be quoted saying “Art is how we decorate space, music is how we decorate time”. I tried to design this web page in the spirit at that idea.

This idea is based off György Ligeti's "Poème symphonique" (1962) which is a musical composition made for metronomes. For more information on the piece, YouTuber Adam Neely has a fun video about it which can be found here. In order to get a more instantaneous more of an instant payload, the notes by default spell out an B7sus2 chord and the pendulums initial starting positions and mass are randomized.

The script makes use of both the JQuery and ToneJS libraries. When testing the web page on Chrome, I found that ToneJS refuses to make sound until a user gesture is made. In order to fix this issue, I implemented and simple splash button which the user has to click in order to start the script.
